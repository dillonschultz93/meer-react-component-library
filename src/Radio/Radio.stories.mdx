# Radio

The `<Radio />` component is an equivalent to the typical `<input type="radio">` element. It should be used to collect single values for submission 
in a form.

<br />
<br />

<Canvas>
  <Story name='Default'>
    {() => {
      const [checkedState, setCheckedState] = useState(false);
      return <Radio id="default" name="default" label="Default" value="default" checked={checkedState} onChange={(e) => setCheckedState(e.target.checked)} />
