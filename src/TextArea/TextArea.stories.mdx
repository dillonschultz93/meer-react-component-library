# TextArea

The `<TextArea>` component is a typical `<textarea>` element. It should be used to collect long forms of data, such as a comment or feedback.

<br />
<br />

The `<TextArea>` component takes a few required props to be properly setup. The `id` prop sets the `id` and `for` attributes on the native `<textarea>`
and `<label>` elements. The `name` prop sets the `name` attribute on the native `<textarea>` element.

<br />

<br />

## Character limit

To set a character limit on the component the `showCount` and `maxLength` props must be passed. `showCount` must be set to `true` before `maxLength` is set.
The `maxLength` prop sets the `maxlength` attribute on the native `<textarea>` element.

<Canvas>
  <Story name='Character Limit'>
    {() => {
        const [value, setValue] = useState('');
        const handleOnChange = (e) => setValue(e.target.value);
        return (
          <div style={{width: '50%'}}>
            <TextArea id="character-limit" name="character-limit" label="Tweet" value={value} onChange={handleOnChange} showCount maxLength={240} />
          </div>
        );
    }}
  </Story>
</Canvas>
