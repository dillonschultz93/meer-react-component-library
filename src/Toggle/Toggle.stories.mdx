# Toggle

The `<Toggle />` component is just a glorified and stylized checkbox.

## Default

<Canvas>
  <Story name='Default'>
    {() => {
      const [checkedState, setCheckedState] = useState(false);
      const handleOnChange = () => {
        setCheckedState(!checkedState);
      }
      return (<Toggle id="toggle" name="toggle" value="toggle" label="Default" checked={checkedState} onChange={handleOnChange} />)
    }}
  </Story>
</Canvas>
