# Button

The `<Button>` component is a typical native `<button>` element or `<a>` element. It should be used to submit forms or to behave as a link.
<br />
<br />

## Role

The role of the `<Button>` component is dependant on the `role ` prop.
<br/>

### Button
The button role can be achieved by passing the value of `'button'` to the `role` prop.

<Canvas>
  <Story name="Role/Button">
    <Button role="button">Button</Button>
  </Story>
</Canvas>

### Submit
The submit role can be achieved by passing the value of `'submit'` to the `role` prop.

<Canvas>
  <Story name="Role/Submit">
    <Button role="submit">Submit</Button>
  </Story>
</Canvas>


### Link
The link role can be achieved by passing the value of `'link'` to the `role` prop. You can then provide a url to the `href` prop and target to the `target` prop. By default the target is set to the value of `'_blank'`.

<Canvas>
  <Story name="Role/Link">
    <Button role="link" href="https://dillonschultz.design">Link</Button>
  </Story>
</Canvas>

<br />

## Prop Definitions

| Name       | Type                                | Value(s)                                      | Optional |
|------------|-------------------------------------|-----------------------------------------------|----------|
| `block`    | `Boolean`                           |                                               | true     |
| `disabled` | `Boolean`                           |                                               | true     |
| `href`     | `String`                            |                                               | true     |
| `icon`     | `React.ReactElement`                |                                               | true     |
| `onClick`  | `(event: React.MouseEvent) => void` |                                               | true     |
| `role`     | `String`                            | 'button' \| 'submit' \| 'link'                | false    |
| `shape`    | `String`                            | 'circle' \| 'round'                           | true     |
| `size`     | `String`                            | 'large' \| 'medium' \| 'small'                | false    |
| `target`   | `String`                            | '_self' \| '_blank' \| '_parent' \| '_top'    | true     |
| `type`     | `String`                            | 'primary' \| 'ghost' \| 'default' \| 'danger' | false    |
